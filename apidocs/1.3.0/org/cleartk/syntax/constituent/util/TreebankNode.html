        public void setTextBegin(int textBegin)
        the offset of the plain text for this node with respect to the sentence text adjusted by the textOffset passed into the parse method. For example, if the entire sentence parse was "(NP-LOC (NNP Calif.) )" and this node corresponded to only "(NNP Calif.)", then getText() would return "Calif." and this method would return 0 plus the textOffset given by TreebankFormatParser.parse(String, String, int).
