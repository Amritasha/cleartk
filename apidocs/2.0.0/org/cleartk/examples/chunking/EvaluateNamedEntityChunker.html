Class EvaluateNamedEntityChunker

  • java.lang.Object
    • org.cleartk.eval.Evaluation_ImplBase<File,AnnotationStatistics<String>>
      • org.cleartk.examples.chunking.EvaluateNamedEntityChunker

  • public class EvaluateNamedEntityChunker
extends Evaluation_ImplBase<File,AnnotationStatistics<String>>

    This class can be used to train and test the named entity chunker. It can be used to either perform 2-fold cross-validation, or training and testing on a holdout test set, or just to evaluate/test a pre-trained model.
