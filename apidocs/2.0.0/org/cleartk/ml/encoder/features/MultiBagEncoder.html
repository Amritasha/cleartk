    public class MultiBagEncoder
extends Object
implements FeatureEncoder<NameNumber>

    Copyright (c) 2009, Regents of the University of Colorado
    All rights reserved. Encodes a Counts feature into a bag-of-words type feature. This works on arbitrary types of objects, not just words. If an object's count is larger than zero, the corresponding value (named using the .toString method of the object) is set to the count. If the count is zero, the value is not included in the output.
