      • getProgress

        public Progress[] getProgress()
        Progress is measured by the number of files in the target directory - not by the number of times getNext has been (and will be) called. This means that the total number of entities to completion is typically going to be 2 or 4 times as many 'documents' that are found depending on what kinds of files exist in the target directory (e.g. *.ag.xml, *.apf.xml, *.sgm, *.tab)
