getTrainingPipeline

List<AnalysisEngine> getTrainingPipeline(String name)
                                         throws UIMAException
Deprecated. 
This method creates a pipeline of analysis engines that will create a training data for the learner being used. This will generally include a CleartkAnnotator that is initialized with a DataWriterFactory. However, a number of other analysis engines may also be required depending on what data from the gold-standard corpus is provided and what needs to be generated.

Parameters:
name - will generally be a value such as 'fold-01' or 'holdout'. It provides a name for this method to provide to the pipeline it returns that serves as a key for that pipeline. One expected use of the name is for naming a subdirectory that a data writer could use to write training data to.

train

void train(String name,
           String... trainingArguments)
           throws Exception
Deprecated. 
This method trains a model(s) for a given training pipeline (identified by name) using whatever training data was generated as a result of running the training pipeline. If your task involves only a single CleartkAnnotator which runs off of a single classification model, then this method will generally consist of a single line of code. However, it is not uncommmon for a task to require many models. In such cases, this method needs to know how to train each model that needs to be trained.

Parameters:
name - will generally be a value such as 'fold-01' or 'holdout'. It provides a name for this method for the pipeline whose data we are training a model(s) for. One expected use of the name is to identify a subdirectory where training data is located.
trainingArguments - arguments that are passed on to the classifiers model trainer.
Throws:
Exception
