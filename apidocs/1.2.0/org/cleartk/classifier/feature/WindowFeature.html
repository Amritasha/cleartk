getStartAnnotation

public static <T extends Annotation> T getStartAnnotation(JCas jCas,
                                                          Class<T> itemClass,
                                                          Annotation focusAnnotation,
                                                          String orientation)
This method returns the annotation at index 0 w.r.t. the orientation of the extractor, the annotation we are extracting features for and the window the provides the boundaries for feature extraction.

Parameters:
jCas - The JCas containing the focus annotation
itemClass - The type of Annotation around the focus annotation to select
focusAnnotation - The annotation from which the orientation is defined.
orientation - The orientation defining which annotation is the start.
Returns:
the annotation at index 0:
  • for ORIENTATION_LEFT return the annotation adjacent and to the left of annotation
  • for ORIENTATION_RIGHT return the annotation adjacent and to the right of annotation
  • for ORIENTATION_MIDDLE return the left most annotation completely contained in annotation, null if does not exist
  • for ORIENTATION_MIDDLE_REVERSE return the right most annotation completely contained in annotation, null if does not exist
