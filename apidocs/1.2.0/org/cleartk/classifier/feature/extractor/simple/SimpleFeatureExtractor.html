List<Feature> extract(JCas view,
                      Annotation focusAnnotation)
                      throws CleartkExtractorException
Some feature extractors will require some specific number of annotations that is greater than one. Such extractors should not implement this interface.

Parameters:
view - the current view of the document.
focusAnnotation - the annotation for which to collect features.
Returns:
