Class CleartkAnnotatorDescriptionFactory

java.lang.Object
  org.cleartk.classifier.CleartkAnnotatorDescriptionFactory
Deprecated. Use AnalysisEngineFactory directly. 

@Deprecated
public class CleartkAnnotatorDescriptionFactory
extends Object

A factory class that simplifies the creation of descriptors for CleartkAnnotator and CleartkSequenceAnnotator classes. The factory methods here wrap common patterns of calls to AnalysisEngineFactory.createPrimitiveDescription(java.lang.Class, org.apache.uima.resource.metadata.TypeSystemDescription, java.lang.Object...) and ConfigurationParameterFactory.addConfigurationParameter(org.apache.uima.resource.ResourceCreationSpecifier, java.lang.String, java.lang.Object) for setting classifier classes, output directories, etc. that are necessary when creating a CleartkAnnotator or CleartkSequenceAnnotator.

Deprecated. Use AnalysisEngineFactory directly.

Create an AnalysisEngineDescription for using a CleartkSequenceAnnotator to write training data for a ViterbiClassifier. Note that data is written with a DataWriterFactory, not a SequenceDataWriterFactory - A ViterbiClassifier wraps a non-sequence classifier to work as a sequence classifier.

Type Parameters:
OUTCOME_TYPE - The outcome type of the classifier (must be the same for both the CleartkSequenceAnnotator and the DataWriterFactory).
Parameters:
annotatorClass - The main annotator for the AnalysisEngineDescription.
typeSystemDescription - The type system used by the annotator.
delegatedDataWriterFactoryClass - The non-sequence DataWriterFactory that will be used to write the training data (and indicate the type of classifier that will be trained from this data).
outputDir - The directory where the training data should be written.
Returns:
An AnalysisEngineDescription for the CleartkSequenceAnnotator.
Throws:
ResourceInitializationException
