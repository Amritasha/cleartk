    public abstract class DirectoryDataWriter<CLASSIFIER_BUILDER extends JarClassifierBuilder<? extends CLASSIFIER_TYPE>,CLASSIFIER_TYPE>
extends Object
    Superclass for DataWriter and SequenceDataWriter implementations that saves files to a training directory using a JarClassifierBuilder. Note that it does not declare that it implements either of the DataWriter interfaces. Subclasses should do this.
