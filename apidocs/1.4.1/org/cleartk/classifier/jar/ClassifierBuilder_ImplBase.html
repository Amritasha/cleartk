    Superclass for builders which write to a training data file using FeaturesEncoders and OutcomeEncoders, and build and package Classifiers as jar files. Subclasses will typically override:
    • EncodingJarClassifierBuilder.saveToTrainingDirectory(File) to add items to the model training directory
    • EncodingJarClassifierBuilder.packageClassifier(File, JarOutputStream) to copy items to the classifier jar
    • EncodingJarClassifierBuilder.unpackageClassifier(JarInputStream) to load items from the classifier jar
    • JarClassifierBuilder.newClassifier() to create a classifier from the loaded attributes
